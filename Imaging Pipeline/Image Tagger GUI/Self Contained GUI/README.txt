TO RUN (in console):
python processGUI.py



Description of Folders and Files:

processorGUI.py:
Main program that will run and help us tag our batches.

batchProcessor.py:
Helper function that processes the different batches of files

exampleBatches:
Folder where all the image batches will go. The program will move batches from this folder into the corresponding folder

organizedBatches:
Folder where all the organized image batches will go. Will have subfolders indicating piece number

Ref Images:
Folder that program uses to draw pictures of pieces. No need to touch anything here.
